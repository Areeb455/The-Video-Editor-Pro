Text Commands

Available Commands:

Basic Operations:
  • trim start=10 end=30 - Trim video from 10s to 30s
  • resize width=1920 height=1080 - Resize to 1920x1080
  • speed factor=2.0 - Change speed (2x faster)
  • extract_audio format=mp3 - Extract audio as MP3
Effects & Color:
  • color_grade preset=cinematic - Apply color preset
  • effect type=blur strength=2 - Apply blur effect
  • overlay type=text content="Hello" position=center - Add text
Basic Operations:
  • trim start=X end=Y
  • resize width=X height=Y
  • speed factor=X
  • extract_audio format=mp3|wav
Color & Effects:
  • color_grade preset=cinematic|vintage|warm|cool|noir|vibrant
  • color_grade brightness=X contrast=Y saturation=Z
  • effect type=blur|sepia|negative|mirror|pixelate|edge_detection strength=X
  • speed_ramp start=X end=Y factor=Z
Animation & Text:
  • animation type=zoom|pan|fade|rotate start=X end=Y scale=Z
  • animation type=pan direction=left|right|up|down
  • overlay type=text content="Text" position=center|top|bottom
  • overlay type=text content="Text" x=X y=Y duration=W
Multi-Video Commands:
  • merge_videos transition=fade duration=2 (requires multiple files)
  • transition type=dissolve duration=1.5 (requires multiple files)
  • Use Multi Video Editor for multi-file operations
Examples:
  • trim start=5 end=15
  • speed factor=1.5
  • color_grade preset=cinematic
  • effect type=blur strength=3
  • overlay type=text content="My Video" position=bottom
